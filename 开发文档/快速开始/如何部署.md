# 如何部署

### 菲比 BOT 技术栈

如果你打算部署类似菲比bot的机器人，你**至少**需要了解以下基础技术栈：

- **Python**
- **Git**
- **Docker**(可选)
- **IP、端口、HTTP/S、WebSocket等网络基础**
- **熟悉Windows 或 Linux至少一种操作系统的终端命令**
- **代理工具(Clash、V2ray等)**

所用到的核心框架文档均整理好，可自行学习部署。
