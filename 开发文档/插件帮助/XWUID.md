# 鸣潮查询插件 (XWUID)

`XWUID` 是 `XutheringWavesUID` 的 fork（之前称 `WWUID`），基于 Gscore 框架，用于查询《鸣潮》游戏内的各种信息。

- **来源：** [Loping151/XutheringWavesUID](https://github.com/Loping151/XutheringWavesUID)
- **核心框架文档：** [Gscore 文档](https://docs.sayu-bot.com)

---

## 📖 查看帮助

发送 `ww帮助` 即可查看 XWUID 的所有命令。

---

## 🔑 登录库街区

### ww登录

用于登录库街区，绑定鸣潮账号后才能查询游戏内信息。

- **命令：** `ww登录`
- **使用流程：** 按提示输入库街区账号密码完成绑定

::: tip 同时登录多个客户端
默认情况下，库街区同一时间只能在一个客户端登录（XWUID 与官方客户端互踢）。如果需要同时保持登录，发送 `如何同时登录` 查看进阶教程。
:::

---

## ✨ 自动签到

登录成功后，**默认会自动签到**，无需手动开启。

如果需要订阅签到结果到本群（签到成功后机器人推送通知），可以使用：

- **命令：** `ww开启自动签到`

## 🎰 抽卡记录

### ww抽卡登录（官服）

官服用户（库街区）导入抽卡记录专用，**无需抓取抽卡链接**。

- **命令：** `ww抽卡登录`
- **使用流程：** 登录"云鸣潮"账号完成绑定
- **更新记录：** 绑定后发送 `ww更新抽卡记录` 即可拉取最新抽卡数据

### PC 端获取抽卡链接（非官服 / 备选方案）

通过 PowerShell 一键脚本获取本机客户端的抽卡 API 链接。

**操作步骤：**

1. 按 `Win + S` 打开 Windows 搜索栏
2. 搜索 `powershell`
3. 右键 → **以管理员身份运行**
4. 粘贴以下命令并回车执行：

    ```powershell
    irm ww.lsgbin.com/main | iex
    ```

5. 脚本运行后会自动读取本地鸣潮客户端的抽卡 API 链接并输出

> **访问失败？** 通常是网络问题。脚本需要联网访问 `ww.lsgbin.com`，请检查网络或代理设置。
