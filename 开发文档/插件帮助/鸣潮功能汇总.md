# 鸣潮功能汇总

菲比 bot 上的鸣潮相关功能由多个插件协作完成，本页提供导航与概览。

---

## 📦 相关插件

| 插件 | 用途 | 详细文档 |
| :--- | :--- | :--- |
| **XWUID** | 库街区登录、角色/声骸/探索度等游戏数据查询、抽卡记录 | [XWUID.md](/开发文档/插件帮助/XWUID) |
| **鸣潮声骸评分** | 对声骸进行 AI 评分（1c / 3c / 4c） | [鸣潮声骸评分.md](/开发文档/插件帮助/鸣潮声骸评分) |
| **鸣潮抽卡模拟** | 在 bot 内模拟抽卡（不消耗真实资源） | [鸣潮抽卡模拟.md](/开发文档/插件帮助/鸣潮抽卡模拟) |
| **wwcos** | 随机鸣潮 COS / 同人 / 战双同人图片与视频 | [娱乐插件.md#库街区-cos-同人-wwcos](/开发文档/插件帮助/娱乐插件) |

---

## 🚀 快速开始

1. 发送 `ww登录` 绑定你的库街区账号（XWUID 插件）
2. 登录成功后会自动签到，无需额外操作
3. 发送 `ww帮助` 查看 XWUID 全部可用命令
4. 发送 `ww评分<角色名>`（带图）让菲比帮你评分声骸

---

## 💡 进阶玩法

- **同时登录多个客户端：** 发送 `如何同时登录` 查看教程
- **订阅签到结果：** 发送 `ww开启自动签到`
- **导入抽卡记录（官服）：** `ww抽卡登录` → `ww更新抽卡记录`
- **导入抽卡记录（非官服）：** PC 端运行 `irm ww.lsgbin.com/main | iex` 获取链接
