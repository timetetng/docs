# 娱乐插件汇总

本页汇总了菲比 bot 上的各类娱乐向插件，涵盖表情包、随机图、查询、抽卡、解释器等玩法。

---

## 🐖 随机猪图

发送 `猪` / `🐷` / `🐖` / `㊗️`（无需 `/`）即可随机获取一张可爱猪猪表情包。

- **图片来源：** [pighub.top](https://pighub.top)

---

## 🐦 随机菲比图

发送 `啾比`（无需 `/`）随机获取一张菲比表情包。

也可以搜索：

- **命令：** `/搜比 <关键词> [搜索数量]`
- **参数：**
    - `关键词`: (必须) 搜索关键词
    - `搜索数量`: (可选) 默认 3
- **图片来源：** [phoebehub.top](https://phoebehub.top)
- **插件仓库：** [timetetng/astrbot_plugin_phoebehub](https://github.com/timetetng/astrbot_plugin_phoebehub)

---

## 🌊 库街区 COS / 同人 (wwcos)

随机发送鸣潮 COS、鸣潮同人、战双同人等图片/视频内容。

| 命令 | 作用 |
| :--- | :--- |
| `wwcos` / `fbcos` / `鸣潮cos` | 随机鸣潮 COS 图片 |
| `wwcos视频` / `fbcos视频` | 随机鸣潮 COS 视频 |
| `ww同人` / `fb同人` / `鸣潮同人` | 随机鸣潮同人图片/视频 |
| `ww同人视频` / `fb同人视频` | 随机鸣潮同人视频 |
| `zs同人` | 随机战双同人图片/视频 |

### 插件更新命令

| 命令 | 作用 |
| :--- | :--- |
| `ww更新cos` / `fb更新cos` | 从 GitHub 更新插件并尝试重载 |
| `ww强制更新cos` / `fb强制更新cos` | 丢弃插件目录本地改动后强制同步远端 |

> 以上命令均无需 `/`。

## 😂 表情包生成器

通过 `bq` 前缀 + 关键词直接触发表情包生成。

- **触发方式：** `bq<关键词>`（无需 `/`）
- **示例：** `bq好耶` `bq震怒`
- **查看帮助：** `/meme帮助`

---

## 🎰 今日 doro 结局

获取今日 doro 随机结局。

- **命令：** `/今日doro`

---

## 🤔 zssm 解释器

引用一条消息并发送 `zssm` / `hyw` / `何意味` 即可让菲比尝试解析并返回解释。

- **触发方式：** 引用消息 + 发送 `zssm` / `hyw` / `何意味`
- **示例：** 引用一段抽象发言 → 发送 `zssm`

---

## 📦 盒

查询指定 QQ 的详细资料卡片。

- **命令：** `盒 <@某人>` 或 `盒 <QQ号>`
- **示例：**
    - `盒 @张三`
    - `盒 12345678`

---

## 🍜 吃什么 / 喝什么

随机推荐美食或饮品。

- **触发方式：** 发送 `吃什么` 或 `喝什么`（无需 `/`）

---

## 💖 今日老婆 / 今日老公 / 今日萝莉

随机发送对应类型美图。

- **触发方式：** 发送 `今日老婆` / `今日老公` / `今日萝莉`（无需 `/`）
- **来源：** [nnlmc/TodayWaifu](https://github.com/nnlmc/TodayWaifu)
