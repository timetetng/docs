# 今日运势 (JRYS)

独立于 Astrbot 框架之外的运势插件，由 [xxuanzQAQ/JRYS](https://github.com/xxuanzQAQ/JRYS.git) 提供。

不同于老签到插件中的运势功能（已停用），JRYS 是独立部署的纯运势系统，不依赖 Astrbot 命令前缀。

---

## ⭐ 触发方式 ⭐

由于 JRYS 不是 Astrbot 插件，命令**不带 `/` 前缀**，直接发送关键词即可触发。

| 命令 | 作用 |
| :--- | :--- |
| `签到` | 抽取今日运势 |
| `运势` | 抽取今日运势 |
| `今日运势` | 抽取今日运势 |
| `逆天改命` | 重新抽取今日运势 |

> 注意：带 `/` 的 `/签到` 仍然会触发老签到插件（用于拿金币），不带 `/` 的 `签到` 才会触发 JRYS 抽取运势。两者互不冲突。

---

## ℹ️ 说明

- 每日可抽取一次今日运势，影响当日的部分互动效果
- `逆天改命` 可消耗一定代价重新抽取一次
- JRYS 与老签到插件的运势系统**互不相通**（独立状态）
